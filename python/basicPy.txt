# Data Types 
# :- Data types defined what type of data a variable can store

# Data type are of two types

# 1) Immutable data types (cannot change)

# 1. String (str)  ->  Store text
b = "hello"
#x[0] = "world"   # ❌

# 2. Integer -> store whole number
a = 5
print(a)

# 3. Float -> store decimal number
num = 4.5
print(num)

# 4. Boolean -> store only two values i.e. true / false
is_logged_in  = True
is_admin = False

# 5. Tuple -> ordered but immutable

c = (1,2,3)
c[0] = 10  #❌


# 2) Mutable data types (can change)

# 1. list (ordered and mutable collection)

fruits = ["orange", "banana"]
fruits[0] = "mango"
print(fruits)

# 2. set -> store unique values
d = {1,2,3,3}
print(d)

# 3. Dictionary (dict) -> store data in key value pair

x = {
    "name" : "A",
    "age" : 22
}











Loops

Loops are used to execute a block of code repeatedly until a condition is met. 
They help avoid writing the same code multiple times and make programs more efficient.

1. For Loop

Used when we know how many times we want to repeat something.

Syntax
for variable in sequence:
    # code


2. While Loop

Used when we don't know exactly how many times the loop should run.

Syntax
while condition:
    # code


3. Infinite Loop
while True:
    print("Hello")

Runs forever because condition is always true.


4. break Statement

Stops the loop immediately.

for i in range(10):

    if i == 5:
        break

    print(i)


5. continue Statement

Skips current iteration.

for i in range(5):

    if i == 2:
        continue

    print(i)

6. pass Statement

Placeholder for future code.

for i in range(5):
    pass

Program runs without error.

7. Nested Loops

Loop inside another loop.

for i in range(3):
    for j in range(2):
        print(i, j)
